## What is open-tools

A cross-platform desktop launcher built with Tauri 2.x + Vue 3 + TypeScript, inspired by Rubick/Alfred/Raycast. Invoked via Alt+Space global shortcut. Plugins are Deno modules distributed via GitHub.

## Commands

```bash
# Frontend dev only
npm run dev

# Full app (Tauri + Vite)
npm run tauri dev

# Build distributable
npm run tauri build

# Download Deno sidecar binary (run once before first build)
bash scripts/download-deno.sh

# Compile plugin-host into Deno sidecar (alternative to download)
bash scripts/build-sidecar.sh
```

## Architecture

Three layers communicate via Tauri IPC (invoke/emit):

**`src/`** — Vue 3 renderer
- `components/SearchBox.vue`: main search input with debounce + keyboard nav
- `components/ResultList.vue`: results display
- `store/app.ts`: Pinia store — `query`, `results`, `loading`; calls `search_apps` and `ping_sidecar` commands

**`src-tauri/src/`** — Rust / Tauri main process
- `lib.rs`: Builder entry — registers all plugins and commands
- `tray.rs`: system tray with Show/Quit menu; left-click toggles window
- `shortcut.rs`: registers Alt+Space to toggle main window
- `sidecar.rs`: spawns `binaries/deno-<triple>`, reads stdout JSON-lines, emits `sidecar-output` events; exposes `ping_sidecar` command
- `db.rs`: SQLite migrations (plugins table + settings table)
- `commands/search.rs`: `search_apps` — scans `.desktop` files on Linux
- `commands/open.rs`: `open_path` — launches apps via Exec= or xdg-open

**`plugin-host/`** — Deno sidecar (TypeScript)
- `main.ts`: reads stdin JSON-lines `{id, type, ...}`, writes stdout JSON-lines responses
- Protocol: `{type:"pong"}` placeholder in Step 1; will dispatch to plugin modules in Step 3

**`src-tauri/binaries/`** — platform-specific Deno binary
- Named `deno-<rust-target-triple>` (e.g., `deno-x86_64-unknown-linux-gnu`)
- Git-ignored; generated by `scripts/download-deno.sh` or `scripts/build-sidecar.sh`

## IPC Protocol (Step 1 placeholder)

```
Vue → invoke("ping_sidecar", {message}) → Rust sidecar.rs → "pong: <message>"
Vue → listen("sidecar-output", handler) ← Rust emit ← Deno stdout JSON-line
```

## Plugin System (planned Step 3)

Plugins are GitHub repos with `rubick.json` metadata. Discovered via GitHub Topics API (`open-tools-plugin`). Installed into `~/.open-tools/plugins/<id>/`. Each plugin runs as a Deno module with declared permissions only.

## Key Files

| File | Purpose |
|------|---------|
| `src-tauri/tauri.conf.json` | Window config, externalBin declaration |
| `src-tauri/capabilities/default.json` | Permission declarations |
| `src-tauri/Cargo.toml` | Rust dependencies |
| `plugin-host/main.ts` | Deno sidecar entry |
